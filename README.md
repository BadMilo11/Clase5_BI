# Dashboards Interactivos para BI: Proyecto Final
### Especialidad en Business Intelligence | Universidad Panamericana (Sede Mixcoac)

Este repositorio contiene la estructura base y el código fuente para el desarrollo de un tablero interactivo utilizando **Streamlit**. Este proyecto se presenta como un ejemplo práctico de las actividades y entregables requeridos para la evaluación final de la especialidad.

---

## 📊 Descripción del Proyecto
El objetivo de este dashboard es facilitar la exploración de datos y la extracción de insights accionables. A través de esta herramienta, se busca transformar conjuntos de datos complejos en visualizaciones dinámicas que apoyen la toma de decisiones estratégicas en el ámbito del Business Intelligence.

## 🛠️ Stack Tecnológico
* **Lenguaje:** Python 3.9+
* **Framework de Dashboard:** Streamlit
* **Manipulación de Datos:** Pandas / NumPy
* **Visualización:** Plotly / Matplotlib / Seaborn
* **Gestión de Dependencias:** Pip
